<br/>

## Installing

1. Clone this repo ```git clone https://github.com/tokyojack/Nasa-Website```
2. Create a ```.env``` file 
3. Add ```REACT_APP_NASA_API_KEY=xxxxxxxxxxxxxx``` to the .env file
4. Run ```npm install```  for the packages
5. Run ```npm start```  to start the program!
